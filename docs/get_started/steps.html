ℹ️

Redirect URL은 제 3자가 우리가 만든 앱을 사용하여 밴드 서비스를 이용할 때 사용합니다.
밴드 OPEN API 서버로 부터 access token을 전달받을 웹 리소스 위치를 의미합니다.
이 예제에서는 임의로 값을 넣어 진행합니다.
