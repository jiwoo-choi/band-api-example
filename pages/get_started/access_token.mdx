# 이해하기

### 인증 토큰(Access Token)

밴드 OPEN API를 사용하기 위해서는 접근하는 정보에 접근할 수 있는 권한을 획득해야 합니다.<br/>
이 권한은 `내 서비스`에 정보를 등록한 이후에 발급되는 <strong>`인증 토큰(access token)`</strong>으로 증명됩니다.

`access token`을 획득하는 가장 쉬운 방법은 `BAND OPEN API` 서비스를 등록하는 것 입니다. <br/>
다음 페이지에서는 [밴드 개발자 사이트](https://developers.band.us)에서 OPEN API 어플리케이션을 만들고 `access token`을 발급 받는 방법을 설명합니다.
