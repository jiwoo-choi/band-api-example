# Access Token 발급 과정

<Steps>
### 개발자 센터 접속하기

[밴드 Open Api 개발자 센터](https://developers.band.us) 사이트에 접속합니다.
<Callout emoji="⚠️">
  `BAND OPEN API 어플리케이션`은 밴드 계정과 연동이 되어야 합니다. <br/>밴드 계정이 없다면 [band](https://band.us) 에서 가입을 먼저 진행해주세요.
</Callout>

### 내 서비스 등록 메뉴 접속
`내 서비스`를 클릭 합니다.

![서비스 등록](/1000.png)

`내 서비스 등록` 버튼을 클릭합니다.

![내 서비스 등록](/2000.png)

### 서비스 등록

서비스 명은 `BAND OPEN API by Perl`로 하고 서비스 유형은 원하는 것을 선택합니다.

![내 서비스 등록](/3000.png)

<Callout type="info" emoji="ℹ️">
  `Redirect URL`은 제 3자가 우리가 만든 앱을 사용하여 밴드 서비스를 이용할 때 사용합니다. 
  <br/> 밴드 OPEN API 서버로 부터 `access token`을 전달받을 웹 리소스 위치를 의미합니다. 
  <br/><strong>이 예제에서는 임의로 값을 넣어 진행합니다.</strong>
</Callout>


### 등록 결과 확인

방금 생성된 `BAND OPEN API by Perl` 애플리케이션은 다음과 같습니다.

![결과목록](/4000.png)
![결과확인](/5000.png)

### 밴드 계정과 서비스 연동

`Access Token` 항목에 <strong>밴드 계정 연동</strong> 을 클릭하면, 해당 애플리케이션을 위한 토큰을 얻을 수 있습니다.

우리가 가진 모든 밴드에 접근 권한을 가진 `access token` 을 만들기 위해, 아래와 같이 가입한 모든 밴드에 권한을 허용합니다.

![결과목록](/6000.png)
![결과확인](/7000.png)

### 🎉  Access token 발급 성공

밴드 계정과 연동된 모든 밴드 접근 권한을 가진 `access token` 이 생성되었습니다. 

![토큰확인](/8000.png)

ZQ 로 시작하고 _xjqf 로 끝나는것이 `access token` 입니다. 

<Callout type="warning">
  이 인증 토큰은 OPEN API를 호출한 사용자가 본인이 맞다는 증명서의 역할을 하므로, <br/>
  타인에게 노출되는 일이 없도록 보안에 유의해주세요.
</Callout>
</Steps>
